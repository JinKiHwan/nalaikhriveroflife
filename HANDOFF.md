주요 요소:

- 우측 중심 교회 십자가 사진과 방향성 그라데이션
- 은은한 몽골 지도 실루엣과 날라흐 포인트

- 히어로 하단에 겹쳐지는 4개의 입체형 바로가기 카드
- 최근 설교, 교회 소식, 예배 안내를 하나의 3열 통합 패널로 구성
- 태블릿과 모바일 반응형 구성
- `prefers-reduced-motion` 대응
- 모바일 가로 넘침 방지를 위해 `.main-area`, `.content-area`, `.home-page`에 `min-width: 0` 적용

기존 링크와 정보 구조는 유지한다.

## 5. 다국어 구조

### 핵심 파일

- `composables/useLanguage.ts`

### 동작 방식

- 지원 언어: 한국어 `ko`, 몽골어 `mn`
- `useCookie('site-language')`로 선택 언어를 저장한다.
- 새로고침과 페이지 이동 후에도 선택값이 유지된다.
- `<html lang>` 값도 선택 언어에 맞게 변경한다.
- 각 컴포넌트에서 다음과 같이 사용한다.

```ts
const { language, t } = useLanguage()
```

```vue
<h1>{{ t('sermons.title') }}</h1>
```

### 현재 번역 적용 범위

- 교회명과 지역명
- 사이드바 메뉴
- 역할명과 로그아웃 문구
- 모바일 상단 교회명
- 홈의 말씀, 바로가기, 최근 설교, 교회 소식, 예배 안내
- 말씀 목록 페이지의 주요 제목, 설명, 빈 상태, 기본 버튼
- 공지 목록 페이지의 주요 제목, 설명, 빈 상태, 기본 버튼
- QT 페이지의 주요 제목, 설명, 빈 상태, 댓글 영역
- 성경 통독 페이지의 주요 제목, 통계 라벨, 안내 및 빈 상태

사용자가 직접 작성한 설교, 공지, QT 본문과 댓글은 자동 번역하지 않고 작성된 원문을 그대로 표시한다.

## 6. 아직 번역이 완전하지 않은 영역

다음 작업자가 우선 확인할 부분이다.

- 말씀 등록 관리자 폼의 세부 라벨과 오류/확인 메시지
- 공지 등록 관리자 폼의 세부 라벨과 오류/확인 메시지
- QT 등록 폼의 세부 라벨과 오류/확인 메시지
- 성경 통독 입력 폼, 피드 문장, 성경 이름 목록
- 말씀/공지/교사방 상세 페이지의 정적 UI 문구
- 교사방 전체 UI
- 관리자 전체 UI
- 별도 인증 레이아웃을 사용하는 로그인 페이지
- 날짜 표기 방식의 언어별 현지화

번역을 추가할 때 문자열을 페이지 안에 직접 분산시키지 말고 `useLanguage.ts`의 메시지 키를 확장하는 방식을 유지한다.
